# Sources

This folder is the canonical location for LaTeX source archives related to the
AffectAI paper track.

Suggested structure:
- `AffectAI_Capture__A_Reproducible_Multimodal_Sensing_Protocol_for_Small_Group_Interaction_Analysis/`
  for the previously submitted protocol paper.
- `auxiliary/` for supplementary source files that are not part of the default
  build from `paper/main.tex`.

Notes:
- Keep the active ICMI submission source in `paper/`.
- Keep participant identifiers anonymized (P1-P4) in all files.
- The existing `auxilary/` folder is kept for backward compatibility; prefer
  `auxiliary/` for new files.
