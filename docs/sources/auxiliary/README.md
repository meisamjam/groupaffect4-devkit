# Auxiliary Sources

This folder stores supplementary LaTeX assets that are not part of the default
submission build from `paper/main.tex`.

Use this folder for:
- Previous protocol-paper source files
- Extended methods text not included in the 8-page main submission
- Backup or alternative figure and table variants
- Notes for camera-ready migration

Guidelines:
- Keep the primary submission source at `paper/main.tex`.
- Avoid changing `paper/main.tex` includes unless you intentionally want
  auxiliary content compiled into the submission PDF.
- Keep participant identifiers anonymized (P1-P4) in all files.
