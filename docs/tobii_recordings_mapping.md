## Recording Folder Structure

Each folder in `C:\Users\meisa\Documents\Tobii` contains:
- `meta/` - Metadata directory
  - `participant` - JSON file with participant ID
  - `RuSerial` - Device serial number
  - `HuSerial` - Head unit serial
  - `RuVersion` - Firmware version
- `recording.g3` - Tobii Glasses 3 binary recording format
- `scenevideo.mp4` - Scene camera video
- `gazedata.gz` - Gaze data (compressed)
- `imudata.gz` - IMU data (compressed)
- `eventdata.gz` - Event data (compressed)
- `snap0.jpg` - Preview snapshot
